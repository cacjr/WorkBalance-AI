WorkBalance AI

Descrição do Projeto:
O WorkBalance AI é um projeto em Python que simula a análise do equilíbrio entre produtividade e bem-estar de colaboradores em um ambiente corporativo.

A aplicação coleta dados diários de trabalho, como horas trabalhadas, pausas, nível de estresse e tarefas concluídas, gerando métricas estatísticas, alertas de risco e feedbacks personalizados para cada colaborador.

Objetivo:
Auxiliar na compreensão do impacto da carga de trabalho sobre a saúde e produtividade dos colaboradores, fornecendo insights que podem apoiar decisões relacionadas a gestão de pessoas, qualidade de vida no trabalho e prevenção de burnout.

Funcionalidades:
Coleta interativa de dados dos colaboradores

Cálculo de métricas estatísticas (média e desvio padrão)

Identificação do colaborador com maior nível de estresse

Detecção de colaboradores altamente produtivos

Geração de alertas de desequilíbrio (alto estresse e poucas pausas)

Feedbacks personalizados com base em estresse e produtividade

Geração e salvamento automático de relatório em arquivo .txt

Tecnologias Utilizadas:
Python 3

NumPy

Jupyter / Terminal

Programação estruturada

Como Executar o Projeto:

Clone este repositório:

git clone https://github.com/seu-usuario/workbalance-ai.git

Acesse a pasta do projeto:

cd workbalance-ai

Execute o script:

python workbalance_ai.py

Preencha as informações solicitadas no terminal.

Saída do Programa:
Exibição de um relatório completo no terminal

Geração do arquivo relatorio_workbalance.txt contendo:

Data e hora da execução

Métricas gerais

Lista de alertas

Feedbacks personalizados
